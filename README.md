## Description

**txt/huffman.txt** 对应小王子英文小说

**bin/encode** 对应编码后的二进制文件

**bin/decode.txt** 对应解码后的英文小说

运行环境: Ubuntu 18.04, g++7.4.0



## How to run

```shell
cd huffman/
make
cd bin
./huffman
```

Please wait for a few seconds.
